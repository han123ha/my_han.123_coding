# 说明/使用

基础移动

1. go'步数'
2. gt(x坐标,y坐标)
3. right'度数'
4. left'度数'
5. penclear''
6. pendown''
7. penup''

循环 

1. for'循环次数'{循环内容}
2. wlile'填‘1’或‘True’可以执行循环内容，反之则不执行循环内容'{循环内容}

变量

1. var'变量名'[对应的值]
2. ---------------↓
3. 可以填算式比如:我的变量+1 提示:(我的变量是一个变量名)
4. ---------------↓
5. 也可以填入字符串比如:'我爱编程' 提示:(字符串必须用英文引号括起)

画布与颜色

1. bcolor'填画笔颜色'提示:(可以填16进制rgb，也可以填red,orange,yellow,green,blue,pink,purple,white,black)
2. color'填画笔颜色'提示:(可以填16进制rgb，也可以填red,orange,yellow,green,blue,pink,purple,white,black)
3. ifr-t/f(填‘1’或‘True’可以设置交互模式为True，反之则交互模式为False)
4. ifrah(设置背景宽度,设置背景高度)
5. ifrxy(设置背景x坐标,设置背景y坐标)
6. iframe[可以填url的网页图片或官网]

打印

1. print[输出内容]提示:(按下esc可以退出输出页面)

条件判断

1. if{填判断条件}{填如果条件满足才执行的程序}
2. else{如果}
